### Built for Developers & Content Creators

Hadron bridges the gap between simplicity for beginners and flexibility for web developers.

* **Beginner Friendly**: Manage content using plain Markdown files or the intuitive Grav Admin panel.
* **Developer Focused**: Custom Twig templating, SCSS support, and extensible blueprints.
* **Multi-Language Support**: Built-in support for multi-language routing and localization out of the box.
