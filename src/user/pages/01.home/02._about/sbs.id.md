### Pengembangan Perangkat Lunak & Arsitektur Sistem

Saya seorang Software Engineer yang berfokus pada penyelesaian tantangan teknis kompleks, penulisan kode yang mudah dipelihara, dan otomatisasi alur kerja penyebaran sistem.

* **Kecerdasan Buatan & Otomatisasi**: Berpengalaman dalam integrasi LLM, sistem Agentic AI, alur kerja RAG, dan orkestrasi agen otonom.
* **Pengembangan Full-Stack**: Menguasai PHP, JavaScript/TypeScript, Python, serta berbagai framework web modern.
* **DevOps & Kontainerisasi**: Keahlian dalam kontainerisasi Docker, optimasi stack LAMP/LEMP, dan konfigurasi server Linux.
* **Desain API & Basis Data**: Mahir dalam merancang RESTful API yang aman, basis data relasional (MariaDB/MySQL), dan sistem caching berkecepatan tinggi (Redis/OPcache).
